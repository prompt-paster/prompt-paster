Native macOS prompt launcher

Prompt Paster

A tiny menu-bar app for keeping reusable coding-agent prompts one shortcut away. Open the overlay, pick a prompt by mouse or keyboard, and paste it wherever you are working.

Download for macOS View source

Requires macOS 14 or newer. Current alpha releases are unsigned until Developer ID signing is enabled.

Local library
